The ODT export backend supports literal examples (see [Literal Examples](/docs/org/Literal-Examples)) with full fontification. Internally, the ODT export backend relies on ‘`htmlfontify.el`’ to generate the style definitions needed for fancy listings. The auto-generated styles get ‘`OrgSrc`’ prefix and inherit colors from the faces used by Emacs Font Lock library for that source language.

For custom fontification styles, customize the `org-odt-create-custom-styles-for-srcblocks` option.

To turn off fontification of literal examples, customize the `org-odt-fontify-srcblocks` option.
