## 13.12 OpenDocument Text Export

The ODT export backend handles creating of OpenDocument Text (ODT) format. Documents created by this exporter use the OpenDocument-v1.2 specification[^1] and are compatible with LibreOffice 3.4.

[^1]: See [Open Document Format for Office Applications (OpenDocument) Version 1.2](https://docs.oasis-open.org/office/v1.2/OpenDocument-v1.2.html).
